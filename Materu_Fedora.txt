2.3 Komponen Sistem Fedora:

Kernel
Fedora Linux adalah sistem oerasi yang menggunakan Linux sebagai kernel-nya untuk berinteraksi dengan hardware. Dikarenakan Linux kernel adalah suatu software yang open source, setiap pengguna sistem operasi berbasis Linux termasuk juga Fedora Linux mampu mengedit source code pada kernel sesuai dengan kehendak pengguna secara gratis.
